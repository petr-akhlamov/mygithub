Как поддержать Ридчарда Столлмана


  1. Заходим на Github, регистрируемся, авторизируемся

  2. Идем на сюда

  3. Нажимаем fork

  4. Идем к себе, ищем у себя этот репозиторий

  5. Идем в _data/signed/

  6. нажимаем Add file> Create New File

  7. Вводим имя Имя.yaml

  8. В содержимое:

    name: Dmitry Meduho

    link: https://github.com/0xf0a00

    где link ссылка на ваш гитхаб

  9. Идем в корень форкнутого репозитория

  10. Жмем Pull Request, затем Create Pull request

  11. Жмем пока добавят подпись
