Как поддержать ридчарда столлмана


  1. 1. Заходим на Github, регистрируемся, авторизируемся

  2. 2. Идем на https://github.com/rms-support-letter/rms-support-letter.github.io

  3. 3. нажимаем fork

  4. 4. Идем к себе, ищем у себя этот репозиторий

  5. 5. Идем в _data/signed/

    нажимаем Add file> Create New File Вводим имя Имя.yaml В содержимое: name: Dmitry Meduho link: https://github.com/0xf0a00 где link ссылка на ваш гитхаб

  6. 6. Идем в корень форкнутого репозитория

  7. 7. Жмем Pull Request, затем Create Pull request

  8. 8. Жмем пока добавят подпись
